Agent Simulation

Paid API skill. Use it after you have an artifact, option, message, post, landing page, or venture concept to test.

Specialized wrappers are included for content checks, message reaction flow, and venture idea simulation.

API

Set your key from the dashboard, then call the simulation endpoint.

export AGENT_SIMULATION_API_KEY="sim_..."

curl -s -X POST "https://agent-simulation-api.vercel.app/api/simulate" \
  -H "content-type: application/json" \
  -H "x-api-key: $AGENT_SIMULATION_API_KEY" \
  -d '{
    "objective": "Decide whether to publish this content.",
    "artifact": {
      "type": "content_draft",
      "content": "Paste the actual text people will see."
    },
    "audience": {
      "description": "Describe who will see this."
    },
    "simulation": {
      "target_n": 40,
      "max_agent_voices": 8,
      "max_output_tokens": 16000
    }
  }'

Exports

Every saved simulation can be exported as Markdown or JSON. Use the export as context in Claude Code, Codex, Cursor, or a team discussion.

curl -s "https://agent-simulation-api.vercel.app/api/simulations/SIMULATION_ID?format=markdown" \
  -H "x-api-key: $AGENT_SIMULATION_API_KEY"
